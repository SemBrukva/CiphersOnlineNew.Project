# Главный вывод

Для CiphersOnline реклама должна рассматриваться как промежуточный источник дохода. Наиболее ценным активом со временем станет аудитория разработчиков и API-доступ к инструментам. Именно API имеет потенциал приносить больше дохода, чем реклама, даже при существенно меньшем количестве пользователей.
